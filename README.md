== Apacerita

Rails is a web-based prototype of a mobile application to gamify travel exploration


== Requirements

Ruby 1.9.3
Rails 3.2.8
MySQL 5.5

== Getting Started

bundle install
bundle exec rake db:create
bundle exec rake db:migrate
bundle exec rails server
